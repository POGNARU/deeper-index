```

```

레포 루트에 두고, 모든 HTML의 `<head>`에 상대경로로 링크한다. 하위 폴더 문서는 `../favicon-32.png`처럼 `../`를 붙인다 — GitHub Pages 프로젝트 사이트는 `/레포명/` 아래로 서빙되므로 루트 절대경로를 쓰면 안 된다.
